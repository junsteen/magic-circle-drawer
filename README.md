# 🔮 Arcane Tracer

**「詠唱の正確さが威力になる」**

スマホ画面上で魔法陣を指でなぞり、その正確さをスコア化するWebアプリケーション。
PWA対応で、スマホにインストールしてオフラインでも遊べます！

## コンセプト

- お手本の魔法陣（三角形・五芒星・六芒星・円・複合、またはランダム生成）をなぞって描画する
- 指でなぞった軌跡が光のラインで描画される
- 正確さに応じてスコア（0〜100点）とランク（S/A/B/C）が算出される
- 難易度ごとに制限時間とスコア倍率が変化
- スコアは魔法のダメージ倍率として出力

## 技術スタック

- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Drawing**: HTML5 Canvas API
- **Touch Events**: Pointer Events API (`onPointerDown`/`onPointerMove`/`onPointerUp`)
- **Build**: Static Export
- **Deployment**: Vercel, Cloudflare Pages

## MVP機能

1. 5種類の魔法陣お手本を表示（三角形・五芒星・六芒星・円・複合）
2. スマホのタッチ操作でなぞって描画
3. 描画精度をスコア化（S/A/B/Cランク）
4. 詠唱タイマー（難易度により変動）
5. リセット＆再挑戦＆次の魔法陣へ
6. 難易度選択（EASY/NORMAL/HARD/EXPERT）
7. ランダム生成された無数の魔法陣で遊べる
8. PWA対応でホーム画面にインストール可能

## 主な機能一覧

- **基本機能**: 魔法陣なぞり描画、精度スコアリング、ランク評価
- **音声入力**: 魔法陣完成ボタンを声の発生検知で自動発動（ボタンは残す）
- **PWA対応**: オフライン利用、ホーム画面インストール、スプラッシュスクリーン
- **魔法陣パターン**: 三角形・五芒星・六芒星・円・複合・一重枚目参照等
- **履歴機能**: 描画履歴保存、シェア機能（Web Share API・JSONダウンロード）
- **難易度設定**: EASY/NORMAL/HARD/EXPERT（制限時間とスコア倍率が変動）
- **ランダム生成**: 無数の魔法陣パターンで継続的に遊べる
- **デプロイ対応**: VercelおよびCloudflare Pagesへの静的サイトデプロイ
- **ビルド最適化**: next exportによる静的サイト生成、キャッシュ最適化

## ランク表

| ランク | スコア | ダメージ倍率 | 目安 |
|--------|--------|------|------|
| **S** | 90〜100 | 120% | 完璧な詠唱 |
| **A** | 70〜89 | 100% | 精密な詠唱 |
| **B** | 50〜69 | 70% | 合格ライン |
| **C** | 〜49 | 0% | 要練習 |

### 難易度倍率

ランクのダメージ倍率に × 難易度倍率が適用されます。

| 難易度 | 制限時間 | 倍率 |
|--------|----------|------|
| EASY | 10秒 | ×0.8 |
| NORMAL | 5秒 | ×1.0 |
| HARD | 3秒 | ×1.3 |
| EXPERT | 2秒 | ×1.6 |

## Getting Started

### 開発

```bash
npm install
npm run dev
```

ブラウザで http://localhost:3000 を開く

### 静的ビルド（本番用）

```bash
npm run build
```

ビルドされた静的ファイルが `out` ディレクトリに出力されます（next.config.ts の output: 'export' 設定による）。

### Vercelデプロイ

このプロジェクトはVercelへのデプロイに対応しています。リポジトリを接続するだけで自動デプロイされます。

### Cloudflare Pagesデプロイ

このプロジェクトはCloudflare Pagesへのデプロイに対応しています。

1. リポジトリをCloudflare Pagesに接続
2. ビルドコマンド: `npm run build`
3. ビルド出力ディレクトリ: `out`
4. プルリクエストごとにプレビュー環境が自動生成されます

## 操作方法

### 初回プレイ

アプリを初めて開くと**チュートリアル**が表示されます。
4ステップで操作方法・スコアリング・お手本アニメーションを確認できます。

### ゲームの流れ

1. 画面上部の **難易度セレクター**（EASY/NORMAL/HARD/EXPERT）をタップして選択
2. **キャンバス上の赤い点**から指でなぞり始める（タイマーが起動）
3. お手本の魔法陣に沿って一周描画
4. 描き終わったら **「詠唱完了！」** ボタンでスコア判定
5. 結果画面でランクとスコアを確認
6. **「リセット」** で同じ魔法陣を再挑戦、**「次の魔法陣 →」** でランダム生成された新しい魔法陣に挑戦

### ヘルプ

右上の **「？」** ボタンをタップすると、ヘルプモーダルが表示されます。
いつでも操作方法を確認できます。

## タッチ対応

- `onPointerDown` / `onPointerMove` / `onPointerUp` を使用
- iPhone Safari・Android Chrome で動作確認済み
- `touch-action: none` でスクロール干渉を防止
- PWAモードでもタッチ操作が正常に動作
